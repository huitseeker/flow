# Introduction

The contracts used in the Cluster Quorum Certificate (QC) and Distributed Key Generation (DKG) protocols
emit events whenever an important action is performed.

These events can be queried using the Go or JavaScript SDKs to extract useful notifications and information about the
state of these processes.
